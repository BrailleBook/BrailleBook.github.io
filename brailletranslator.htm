Braille Dataset

To make a machine learning system that can recognize Braille characters and translate them into English, we first collected a dataset of Braille characters divided as our training and validation datasets. The data contains 1560 pieces of Braille images. The dataset is classified into 26 subsets with each represents a English letter. All Braille images in the same subset represents the same English letter. The data source can be found in kaggle: https://www.kaggle.com/datasets /shanks0465/braille-character-dataset.

First Step: Train a Machine Learning Model

In the video, I walked through the steps of creating a Braille recognizer model. After using two deep learning models, resizing the training set of Braille images, and training it for a dozen of times, the accuracy of the predictions reached 98%!
